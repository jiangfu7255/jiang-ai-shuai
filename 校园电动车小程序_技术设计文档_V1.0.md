## 一、项目概述

### 1.1 项目背景
本项目为校园专属电动车租赁 + 上门换电轻资产服务小程序，面向高校学生提供租车配送、上门换电、电池回收充电的完整闭环服务。

### 1.2 核心功能模块
| 模块 | 功能说明 |
|------|----------|
| 用户模块 | 实名注册、学生认证、登录、套餐订阅 |
| 租车模块 | 车辆展示、套餐购买、订单管理 |
| 换电模块 | 申请换电、定位上报、换电记录 |
| 电池模块 | 电池状态管理、回收充电 |
| 增值服务 | 定位查询、电子围栏、轨迹回放、故障报修 |
| 管理后台 | 订单调度、车辆管理、电池管理、兼职管理 |

## 四、用户使用流程

### 4.1 整体流程图

```
┌─────────────────────────────────────────────────────────────┐
│                        用户端流程                             │
└─────────────────────────────────────────────────────────────┘

[注册/登录] → [学生认证] → [选择套餐] → [支付下单]
                                    ↓
                            [等待车辆配送]
                                    ↓
                            [确认收货开始用车]
                                    ↓
        ┌─────────────────────────────┼─────────────────────────────┐
        ↓                             ↓                             ↓
   [日常出行]                   [电量不足]                   [故障/事故]
        │                             │                             │
        ↓                             ↓                             ↓
   [使用车辆]              [申请上门换电]              [提交故障报修]
        │                             │                             │
        ↓                             ↓                             ↓
   [骑行结束]              [兼职人员上门换电]          [等待维修处理]
        │                             │                             │
        └─────────────────────────────┼─────────────────────────────┘
                                     ↓
                              [换电/服务完成]
                                     ↓
                              [继续日常使用]
```

### 4.2 租车流程（详细）

```
1. 用户打开微信小程序
2. 点击"注册"，填写手机号、密码、姓名、学号
3. 上传学生证照片提交认证
4. 认证通过后，进入首页
5. 点击"租车"，查看可选套餐：
   - 基础租车月卡：128元/月
   - 租车学期套餐：499元/学期
   - 爆款组合套餐：169元/月（租车+无限次换电）
6. 选择套餐并支付
7. 填写配送地址和期望送达时间
8. 提交订单，等待后台调度
9. 兼职人员取车并配送到指定位置
10. 用户确认车辆外观、电量等
11. 点击"确认收货"，租车生效
12. 开始使用电动车
```

### 4.3 上门换电流程（详细）

```
1. 用户骑行中，小程序首页显示电量低于20%警告
2. 用户点击"申请换电"
3. 系统自动获取车辆GPS位置
4. 用户确认换电地点（可微调）
5. 提交换电申请
6. 后台收到申请，调度附近兼职人员
7. 兼职人员接单，前往用户位置
8. 兼职人员到达后，拆卸亏电电池
9. 更换为满电电池
10. 兼职人员在小程序点击"换电完成"
11. 系统自动扣除套餐次数或单次费用（5元/次）
12. 兼职人员将亏电电池带回校外充电点
13. 用户收到换电完成通知，继续出行
```

### 4.4 增值服务流程

#### 4.4.1 车辆定位
```
1. 用户在小程序点击"车辆位置"
2. 展示车辆当前实时位置（地图）
3. 可查看GPS定位更新时间
```

#### 4.4.2 电子围栏
```
1. 后台设置校园范围电子围栏
2. 车辆驶出围栏区域
3. 系统自动推送提醒通知到用户小程序
4. 用户确认是否为本人驾驶外出
```

#### 4.4.3 故障报修
```
1. 用户点击"故障报修"
2. 选择故障类型（轮胎/刹车/其他）
3. 描述故障情况，可上传图片
4. 提交报修单
5. 后台收到通知，调度维修人员
6. 维修人员联系用户，约定处理时间
7. 维修完成后用户确认
```
